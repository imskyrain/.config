# .config

个人配置文件集合，包含常用工具的配置。

## 包含的配置

- **tmux** - 终端复用器配置
- **nvim** - Neovim 编辑器配置
- **yazi** - 文件管理器配置
- **alacritty** - 终端模拟器配置

## 安装

```bash
# 克隆仓库
git clone git@github.com:imskyrain/.config.git ~/.config

# 或者克隆到指定位置并创建符号链接
git clone git@github.com:imskyrain/.config.git ~/dotfiles/.config
ln -s ~/dotfiles/.config ~/.config
```
