# 快速参考卡片 🚀

## 核心快捷键（已智能优化）

### 文件操作

```
新建文件：
  Ctrl+N        提示输入文件名，立即创建 ✨

保存文件：
  Ctrl+S        未命名文件自动提示命名 ✨
  :w            传统保存命令

关闭文件：
  Ctrl+W        智能提示，防止丢失内容 ✨
  :q            退出当前窗口
  :qa           退出所有窗口
  :q!           强制退出（丢弃修改）

切换文件：
  Ctrl+Tab      下一个文件
  Ctrl+Shift+Tab 上一个文件
```

✨ = 智能行为，详见 `SMART_FILE_MANAGEMENT.md`

---

### 文件浏览

```
Oil 文件浏览器（推荐）：
  <leader>e     打开浮动窗口
  回车          打开文件/进入目录
  -             返回上级目录
  dd + :w       删除文件
  yy + p + :w   复制文件
  cw + :w       重命名文件
  o + :w        创建新文件
  g?            显示帮助

Telescope 模糊查找：
  <leader>ff    查找文件
  <leader>fg    查找内容
  <leader>fb    查找 buffer
```

---

### 编辑模式

```
进入编辑：
  i             在光标前插入
  a             在光标后插入
  o             在下方新建行
  O             在上方新建行

退出编辑：
  Esc           返回普通模式
  Ctrl+[        返回普通模式（替代）
```

---

### Session 管理

```
自动功能：
  ✅ 退出时自动保存状态
  ✅ 启动时自动恢复状态

手动控制：
  <leader>qs    恢复当前目录 session
  <leader>ql    恢复上次 session
  <leader>qd    停止保存 session
```

## 工作流程示例

### 场景 1：创建新文件

```
方法 1（快速）：
  1. Ctrl+N
  2. 输入：src/utils/helper.js
  3. 开始编辑
  4. Ctrl+S 保存

方法 2（浏览式）：
  1. <leader>e
  2. 导航到 src/utils/
  3. o（新建行）
  4. 输入：helper.js
  5. :w（保存）
  6. 回车打开
```

### 场景 2：编辑多个文件

```
1. <leader>e 打开 Oil
2. 浏览并打开文件 1
3. <leader>e 再次打开 Oil
4. 打开文件 2
5. Ctrl+Tab 在文件间切换
6. Ctrl+S 保存当前文件
7. Ctrl+W 关闭当前文件
```

### 场景 3：查找并编辑

```
1. <leader>ff
2. 输入文件名部分
3. 回车打开
4. 编辑内容
5. Ctrl+S 保存
6. :wqa 保存所有并退出
```

## 智能保护机制 🛡️

### Ctrl+N - 新建文件

```
旧行为：创建空白 buffer
新行为：立即提示输入文件名

示例：
  Ctrl+N → 输入 "test.js" → 创建 test.js
  Ctrl+N → 取消输入 → 不创建任何东西
```

### Ctrl+S - 保存文件

```
已命名文件：直接保存
未命名文件：提示输入文件名

示例：
  已命名：Ctrl+S → 保存成功
  未命名：Ctrl+S → "保存为: _" → 输入文件名
```

### Ctrl+W - 关闭文件

```
无修改：直接关闭
已命名有修改：Vim 提示
未命名有修改：三选一对话框
  1. 保存     → 提示文件名
  2. 不保存   → 丢弃内容
  3. 取消     → 继续编辑
```

### 退出检查

```
未命名且有内容 → 阻止退出 → 提示保存

绕过方法：
  :q!         强制退出（丢失内容）
  :qa!        强制退出所有
```

## 最常见问题 FAQ

### Q: 怎么退出 Neovim？

```
A: 按 Esc 确保在普通模式，然后：
   :q          退出
   :wq         保存并退出
   :q!         强制退出（不保存）
```

### Q: Ctrl+N 后如何取消？

```
A: 在文件名输入框按 Esc 或不输入直接回车
   不会创建任何文件
```

### Q: 未命名文件如何保存？

```
A: 按 Ctrl+S，会自动提示输入文件名
```

### Q: 如何强制关闭不保存？

```
A: 在关闭对话框选择"不保存"
   或使用 :bd! 强制删除 buffer
```

### Q: 启动很慢怎么办？

```
A: 查看 OPTIMIZATION.md
   首次启动需要安装插件（1-3分钟）
   后续应该很快（1-3秒）
```
