# Sublime Text 风格快速入门 🚀

## 立即开始使用

### 第一步: 安装插件

```vim
:Lazy sync
```

重启 Neovim 后即可使用所有功能。

---

## 🔥 最常用的快捷键

### Session 管理（自动保存/恢复）

```
自动功能：
✅ 退出时自动保存工作状态
✅ 启动时自动恢复上次状态

手动恢复：
<leader>qs    恢复 Session
<leader>ql    恢复上次 Session
```

### Oil 文件浏览器

```
打开方式：
<leader>e     打开浮动文件浏览器 ⭐ 推荐
<leader>E     在当前窗口打开
-             打开父目录

基本操作：
<CR>          打开文件/进入目录
-             返回上级目录
g.            显示/隐藏隐藏文件
<C-c>         关闭

编辑文件系统：
dd            删除文件
yy + p        复制文件
cw            重命名文件
o             创建新文件/目录
:w            保存所有修改 ⚡ 重要
```

### Sublime Text 快捷键（已智能优化）

```
<C-n>         新建文件（立即提示输入文件名）✨
<C-s>         保存文件（未命名文件自动提示）✨
<C-w>         关闭文件（未保存时智能提示）✨
<C-Tab>       下一个文件
<C-a>         全选

✨ = 智能行为，详见 SMART_FILE_MANAGEMENT.md
```

## 📖 使用场景

### 场景 1: 项目开发（Session 恢复）

```bash
# 第一天
cd ~/my-project
nvim
# 打开多个文件编辑...
:wqa

# 第二天
cd ~/my-project
nvim
# 🎉 自动恢复！所有文件都在！
```

### 场景 2: 文件管理（Oil）

```
1. 按 <leader>e 打开 Oil
2. 导航到文件位置
3. 像编辑文本一样管理文件：
   - dd 删除
   - yy 复制
   - cw 重命名
   - o 新建
4. :w 保存修改
```

### 场景 3: 快速新建文件

```
方法 1 - 使用 Oil:
<leader>e     打开 Oil
o             新建行
输入文件名     例如: "test.js"
:w            保存创建

方法 2 - 直接新建:
<C-n>         新建空白文件
<C-s>         保存并命名
```

## 💡 5 个必知技巧

### 1. Oil 中批量重命名

```vim
" 在 Oil 中
:%s/old_name/new_name/g
:w
" 所有匹配的文件都被重命名了！
```

### 2. 撤销 Oil 操作

```
在保存前：按 u 撤销
保存后：
- 删除的文件在回收站
- Git 项目可以用 git 恢复
```

### 3. Oil 预览文件

```
<C-p>         预览文件内容（不打开）
上下键        切换预览不同文件
```

### 4. 快速组织文件

```vim
" 在 Oil 中
o            创建 "new_folder/" 目录
dd           剪切文件
<CR>         进入新目录
p            粘贴文件
:w           应用移动
```

### 5. 分屏打开文件

```
在 Oil 中：
<C-s>        垂直分屏打开
<C-h>        水平分屏打开
<C-t>        新标签页打开
```

## 🎯 第一次使用建议

1. **试用 Session**
   ```bash
   cd ~/test-project
   nvim file1.txt
   :e file2.txt
   :wqa
   # 重新打开
   nvim
   # 看！文件都恢复了！
   ```

2. **试用 Oil**
   ```vim
   <leader>e
   " 使用 j/k 浏览
   " 按回车打开文件
   " 按 g? 查看所有快捷键
   ```

3. **常用工作流**
   ```
   <leader>e     浏览文件
   回车          打开文件
   <C-Tab>       切换文件
   <C-s>         保存
   :wqa          退出（自动保存 session）
   ```
