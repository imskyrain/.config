# 🚀 快速开始指南

## 第一次使用 Neovim

### 1️⃣ 安装插件（首次必须）

```bash
# 启动 Neovim
nvim

# 等待 lazy.nvim 自动安装插件
# 你会看到一个窗口显示下载进度
# ⏳ 等待所有插件安装完成（约1-3分钟）
# ✅ 看到所有绿色勾或 "Done" 后按 q 退出
```

### 2️⃣ 基本使用

```bash
# 进入项目目录
cd ~/your-project

# 启动 Neovim（不带参数）
nvim

# 现在你在 Neovim 里了！
```

### 3️⃣ 最重要的快捷键

```
正常模式下（默认模式）：
  <leader>e     打开文件浏览器（leader = 空格键）
  :q            退出
  :w            保存
  :wqa          保存所有并退出

插入模式（编辑文字）：
  按 i          进入插入模式
  按 Esc        退出插入模式
  <C-s>         保存文件（Ctrl+s）
```

### 4️⃣ 浏览和打开文件

```vim
" 方法1：使用 Oil 文件浏览器（推荐）
空格键 + e      " <leader>e 打开浮动文件浏览器
j/k            " 上下移动
回车           " 打开文件或进入目录
-              " 返回上级目录
q              " 关闭

" 方法2：使用 Telescope 查找
空格键 + f + f  " <leader>ff 模糊查找文件
输入文件名部分
回车打开
```

## 🎯 核心功能

### Session 自动恢复（类似 Sublime Text）

```bash
# 第一天
cd ~/my-project
nvim
# 打开多个文件编辑...
:wqa

# 第二天
cd ~/my-project
nvim
# 🎉 所有文件自动恢复！
```

### Oil 文件管理（像编辑文本一样管理文件）

```vim
" 打开 Oil
<leader>e

" 操作文件（就像编辑文本）
dd            " 删除文件
yy + p        " 复制粘贴文件
cw            " 重命名文件
o             " 创建新文件
:w            " 保存所有修改

" 导航
<CR>          " 打开文件/进入目录
-             " 返回上级
g.            " 显示/隐藏隐藏文件
g?            " 显示所有快捷键
```

## 📖 常见操作

### 编辑文件

```vim
" 打开文件
<leader>e       " 浏览并选择文件
<C-n>           " 新建空白文件

" 编辑
i               " 进入插入模式（开始编辑）
Esc             " 退出插入模式
<C-s>           " 保存

" 在多个文件间切换
<C-Tab>         " 下一个文件
<C-S-Tab>       " 上一个文件
<C-w>           " 关闭当前文件
```

### 保存和退出

```vim
" 保存
:w              " 保存当前文件
<C-s>           " 快捷键保存

" 退出
:q              " 退出当前窗口
:qa             " 退出所有窗口
:wq             " 保存并退出
:wqa            " 保存所有并退出
:q!             " 强制退出（不保存）
```

### 查找文件和内容

```vim
" Telescope 模糊查找
<leader>ff      " 查找文件
<leader>fg      " 查找内容（grep）
<leader>fb      " 查找 buffer
<leader>fh      " 查找帮助

" 在 Telescope 中：
↑↓ 或 Ctrl+j/k  " 上下移动
<CR>            " 打开选中项
<Esc>           " 取消
```

## ⚠️ 重要提示

### 启动问题

**如果启动时卡住**:
1. 等待插件安装完成（首次可能需要3-5分钟）
2. 看到 lazy.nvim 窗口按 `q` 退出
3. 如果一直卡住，按 `Ctrl+C` 退出，然后查看 `OPTIMIZATION.md`

**避免启动卡顿**:
- ✅ 已禁用 DAP 调试插件（导致卡顿的主要原因）
- ✅ 已禁用 AI 插件
- ✅ 预期启动时间：1-3秒

### 获取帮助

```vim
" 在 Neovim 中
:help           " 打开帮助
<leader>        " 按空格键等待，显示所有快捷键
g?              " 在 Oil 中显示帮助
```

**查看文档**:
- `SUBLIME_QUICKSTART.md` - 5分钟快速上手
- `SUBLIME_MODE.md` - 完整功能指南
- `OPTIMIZATION.md` - 优化和问题排查

## 🎓 学习路径

### 第1天：基础操作
1. 学会进入/退出 Neovim (`:q`, `:wq`)
2. 学会插入模式和正常模式（`i`, `Esc`）
3. 学会保存文件（`:w` 或 `<C-s>`）
4. 学会用 Oil 浏览文件（`<leader>e`）

### 第2天：文件管理
1. 熟练使用 Oil 打开文件
2. 学会在多个文件间切换（`<C-Tab>`）
3. 学会创建和关闭文件（`<C-n>`, `<C-w>`）
4. 体验 Session 自动恢复

### 第3天：高效操作
1. 使用 Telescope 快速查找（`<leader>ff`）
2. 用 Oil 管理文件（dd, yy, p, cw）
3. 学习更多 Vim 移动命令（h, j, k, l, w, b）

### 之后：深入学习
- 学习 Vim 的各种模式和命令
- 自定义配置和快捷键
- 探索更多插件功能

## 🆘 求助

### 我卡在了 lazy.nvim 界面

```
按 q 退出
如果不行，按 :q 然后回车
如果还不行，按 Ctrl+C
```

### 我不知道怎么退出 Neovim

```
按 Esc 确保在正常模式
输入 :q 然后回车

如果提示未保存：
:q!  强制退出
:wq  保存并退出
```

### 启动很慢或卡住

查看 `OPTIMIZATION.md` 文档

### 插件安装失败

```vim
:Lazy clean
:Lazy sync
```
